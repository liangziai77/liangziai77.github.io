#### github+hexo个人博客
```bash
hexo g	#生成静态文件
hexo s	#启动web本地预览
hexo d	#部署提交至github
hexo clean	#清除缓存文件 (db.json) 和已生成的静态文件 (public)
```

#### 开启自启动程序
```
win+R 打开运行框  
输入: shell:startup  
将程序快捷方式拖入即可  
```

#### ping命令
```bash
ping -n num IP地址    #发送num个IP包
ping -t IP地址        #不停发送IP包，Ctrl+C停止
ping -l size IP地址   #发送size大小的IP包
ipconfig              #查看IP地址
ipconfig -all         #查看网卡详细，包括MAC地址
```

#### 基本命令
```
·命令：color f0
  帮助：color ?
  作用：改变背景及字体颜色
·命令：cls
  作用：清屏
·命令：shutdown -s -t 00
  作用：关闭电脑
·命令：shutdown -r -t 00
  作用：重启电脑
·命令：shutdown -a
  作用：关机
·命令：shutdown -l
  作用：锁屏
·命令：shutdown -h
  作用：显示关机时间
```
